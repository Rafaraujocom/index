    • Projeto de Educação Financeira:

    O Projeto de Educação Financeira visa proporcionar aos participantes o conhecimento necessário para tomar decisões financeiras mais conscientes, equilibradas e responsáveis. O projeto integra diversas disciplinas, desenvolvendo não apenas o entendimento sobre finanças pessoais, mas também habilidades práticas para gerenciar dinheiro de forma eficiente, planejar investimentos e evitar armadilhas financeiras

    Objetivo do Projeto:

    Este projeto busca preparar os alunos para o manejo financeiro do dia a dia, desde o planejamento do orçamento pessoal até o entendimento sobre investimentos e poupança. Além disso, abordamos temas como consumo responsável, endividamento, e a importância de uma boa gestão financeira na vida cotidiana Disciplinas Envolvidas no Projeto

    Disciplinas Envolvidas:

    Disciplinas Envolvidas O projeto envolve uma abordagem multidisciplinar para garantir uma compreensão profunda e prática do tema. As disciplinas envolvidas são: Matematica,Portugês,projeto de desing web,desenvolvimento de front-end

    Matemática e Design:

    Matemática: Ensina cálculos financeiros, como juros simples e compostos, porcentagens e planejamento de orçamentos. Português: Foca na interpretação de documentos financeiros e na redação de relatórios claros sobre finanças. Projeto de design Web: Desenvolve interfaces digitais acessíveis para ferramentas de educação financeira, priorizando a experiência do usuário. Desenvolvimento Front-End: Ensina a programar e implementar interfaces digitais usando HTML, CSS e JavaScript, garantindo responsividade em diferentes dispositivos.
